Buena pillada. Esa línea ya no encaja con la estructura nueva que te propuse, porque ahora el dataset vive fuera de project/, no dentro. Vamos a verlo.

El problema
Con la nueva estructura:

CV-II-PROJECT/
├── monuseg_dataset/         ← el dataset está aquí
└── project/
    ├── Project_Hugo_Adrian.ipynb
    └── src/
Cuando ejecutas el notebook desde project/, Path.cwd() te da .../CV-II-PROJECT/project/. Por tanto Path.cwd() / "MoNuSeg" busca el dataset en .../CV-II-PROJECT/project/MoNuSeg/ — que no existe.

La solución limpia
Cambiar DATASET_DIR para que suba un nivel y busque al lado de project/:

python
PROJECT_ROOT = Path.cwd()
DATASET_DIR = PROJECT_ROOT.parent / "monuseg_dataset"
Esto asume la estructura que te propuse. Si Hugo prefiere tener el dataset en otro sitio (por ejemplo dentro de project/ para mantener la convención original), se ajusta a lo que tengáis.

Mi recomendación: hacerlo configurable
En lugar de hard-codearlo, lo más flexible es usar una variable de entorno con un valor por defecto sensato:

python
import os
from pathlib import Path

PROJECT_ROOT = Path.cwd()

# Allow overriding the dataset location via an environment variable.
# Default: assume the dataset lives next to this project folder, in
# ../monuseg_dataset/ relative to the notebook.
DATASET_DIR = Path(
    os.environ.get("MONUSEG_DATASET_DIR", PROJECT_ROOT.parent / "monuseg_dataset")
)

if not DATASET_DIR.exists():
    raise FileNotFoundError(
        f"Dataset not found at {DATASET_DIR}. "
        f"Either place the MoNuSeg images there, or set the MONUSEG_DATASET_DIR "
        f"environment variable to the correct path."
    )

print(f"Project root: {PROJECT_ROOT}")
print(f"Dataset dir:  {DATASET_DIR}")
Ventajas:

Funciona "out of the box" si seguís la estructura recomendada.
Hugo puede ejecutarlo con su layout actual sin tocar nada haciendo export MONUSEG_DATASET_DIR=/ruta/que/sea antes de lanzar Jupyter.
Da un mensaje de error claro si la ruta no existe, en lugar de un críptico RuntimeError: No images with extension *.png were found desde utils.py.
Los profesores que evalúen el notebook ven inmediatamente dónde poner el dataset.
